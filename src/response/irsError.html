Pengisian IRS - Anda tidak dapat mengisi IRS

Keterangan

Anda tidak dapat mengisi IRS karena periode registrasi akademik belum dimulai atau sudah berakhir.

Registrasi akademik dimulai pada pukul 09.00 Waktu Server Mimi-kyu pada tanggal mulai registrasi akademik dan berakhir setelah lewat pukul 23.59 Waktu Server Mimi-kyu pada tanggal berakhir registrasi akademik.

Solusi

Hubungi bagian akademik di program studi/fakultas Anda.


Anda tidak dapat mengubah IRS karena statusnya telah disetujui.

Solusi

  1. Hubungi pembimbing akademis Anda untuk mengubah status IRS.
  2. Mengubah IRS Anda pada periode add/drop
  3. Hubungi bagian akademik di program studi/fakultas Anda.
